## Cara Login

Anda dapat login menggunakan dua metode:

1. **Username**: Gunakan nama pengguna Anda (misal: `kasir1`).
2. **NIS (Nomor Induk Siswa)**: Masukkan nomor induk Anda jika Anda adalah staf siswa (misal: `12345`).
   _Sistem akan mendeteksi otomatis jenis login Anda._

## Memproses Transaksi

1. **Tambah Produk**: Klik gambar produk di layar atau gunakan **Barcode Scanner**.
2. **Atur Jumlah**: Gunakan tombol **+** atau **-** pada keranjang belanja.
3. **Checkout (F2)**: Tekan tombol Checkout, masukkan nominal uang tunai yang diterima.
4. **Konfirmasi**: Sistem akan menampilkan jumlah kembalian. Pastikan uang sudah benar.
5. **Cetak Struk**: Klik tombol **Cetak Struk** hanya jika pelanggan memintanya. Anda bisa menutup jendela tanpa mencetak.

## Shortcut Keyboard (Update 2026)

Untuk mempercepat pelayanan secara signifikan tanpa mengandalkan klik mouse, kasir dapat menggunakan sistem **Shortcut Keyboard Global**. Shortcut ini dapat diakses kapan saja:

- **F1**: Membuka **Panduan Shortcut** (Bantuan).
- **F2**: Langsung menuju proses **Bayar / Checkout**.
- **F4**: Langsung memfokuskan kursor ke **Input Barcode Scanner** dan membersihkan isinya sebelumnya.
- **F8**: Membuka daftar **Transaksi Tertunda (Held Transactions)**.
- **F9**: Menunda transaksi yang sedang berjalan (Hold).
- **Alt + C**: Menghapus seluruh isi keranjang belanja.
- **Esc (Escape)**: Menutup popup yang sedang aktif atau membersihkan kolom pencarian produk.

_Catatan Cerdas: Sistem memiliki fitur **Smart Focus**. Setiap kali transaksi selesai atau popup ditutup, layar akan otomatis mengembalikan fokus ke area scan barcode, sehingga kasir dapat melakukan scan barang antrian selanjutnya tanpa henti._

## Tips Cetak Struk

Jika printer thermal Anda bermasalah:

1. Periksa kabel USB/Koneksi.
2. Saat kotak dialog muncul, pilih **Margin: None** dan **Scale: 100%**.
3. Pastikan **Header & Footer** tidak dicentang agar struk terlihat rapi.
