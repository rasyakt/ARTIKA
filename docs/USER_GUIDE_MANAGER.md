## ðŸ“‹ Fitur Utama Manager

Berbeda dengan Kasir, Manager memiliki otoritas untuk memantau performa toko secara real-time.

### 1. Dashboard Ringkasan

- Memantau omzet harian secara langsung.
- Melihat total transaksi yang sedang berlangsung.
- Grafik produk terlaris.

### 2. Reports Hub (Pusat Laporan)

Manager dapat mengakses dan mencetak laporan dari tiga departemen utama:

- **Warehouse**: Laporan stok masuk/keluar dan nilai aset.
- **Cashier**: Laporan penjualan, metode pembayaran, dan retur.
- **Finance**: Jurnal transaksi otomatis (Debit/Kredit).

### 3. Audit Log (Rekam Jejak)

Fitur pengawasan untuk melihat aktivitas staf:

- Siapa yang melakukan login/logout.
- Perangkat apa yang digunakan (IP & Nama Device).
- Detail aksi krusial (hapus data, update stok).

### 4. Koreksi & Rollback (Penting!)

Jika kasir melakukan kesalahan input yang sudah ter-checkout:

1. Buka menu **Laporan Kasir**.
2. Cari transaksi yang salah.
3. Klik tombol **Items** untuk melihat detail.
4. Klik **Rollback** untuk membatalkan transaksi dan mengembalikan stok secara otomatis.
